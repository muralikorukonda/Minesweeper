1. How to clone the project ?

   git init
   git clone https://github.com/dongxuan/Freecell
   git remote add freecell https://github.com/dongxuan/Freecell
   git fetch freecell

   step1: update your code

   step3: commit your code

       git commit -m "commit message"
   
   step4: push your commit to github.
       git push origin master
